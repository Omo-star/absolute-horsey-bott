## Absolute Horsey Bot
### Activation
1. Click -> [AbsoluteHorseyBot](https://discord.com/channels/@me/1440204239056928880/1441096182427287634)
- Add to all apps
2. Mention it in any server with @ and it will respond! Replying doesn't count.


### Commands
```!roastmode [mode]``` (Enter roast mode)
- ```!roastmode deep``` - (Recommended) Deep mode for better roasts, but a bit slower
- ```!roastmode fast``` - Speedy roasts, but with less punch
- ```!roastmode adjustable``` - Adjusts roast power based on your roasts
- ```!roastmode stop``` - Exit roast mode

```!roast [user(s)] [extra prompt]``` (roast certain user(s) then return to regular mode)

```autor [mode]``` (Auto-roasts whoever pings the bot)
- ```autor on``` - Turn on auto-roast mode
- ```autor off``` - Turns off auto-roast mode
  
```!data [user]``` (Retrieves data about a certain user)
